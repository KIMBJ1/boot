4. 배열
	1. 다량의 데이터를 하나의 변수로 관리하는 구조
		- 개별 데이터 구조는 어떻게 분리는 고유한 index값
		- 0부터 시작
			: 문자열의 각 글자 의미하는 index도 0부터 시작
	2. java에서의 제약 조건
		- 동일한 타입에 한해서만 배열에 저장 및 활용
	3. 기본타입과 참조 타입으로 배열 생성 및 활용
	4. 배열 타입은 참조 타입
	5. 배열 저장 영역 : heap
	6. 문법
		1. [] : 배열 의미하는 표현
		- 선언, 생성, 초기화 동시 표현
			- 타입 [] 변수명 = {값1, 값2, ...};
			- new 없이도 객체 생성(String과 배열만 해당)
		- 선언, 생성, 초기화 분리해서 개발
			- 선언 하는 법 : 타입 [] 변수명;
			- 생성 : 변수명 = new 타입[메모리 갯수]
			- 초기화 : 변수명[0등 index] = 값;
		2. [] = 1차원 / [][] = 2차원 / [][][] = 3차원
	7. length
		- 배열이 메모리에 생성시 자동 생성되는 배열만의 변수
		- 생성된 배열의 메모리 개수

5. 리뷰
	- 데이터 타입 종류
		- 기본 - byte / int / short / long | float / double | char | boolean
			: 값만 표기에 맞게 변수에 대입 활용
		- 참조 - 객체타입
			- api 제공하는 모든 클래스 타입 / 사용자가 개발하는 클래스 타입 / 배열(배열도 타입이다)
			- 객체 생성(new 생성자[...])해서 변수에 대입해서 활용
			- 객체가 보유한 변수와 메소드 호출로 활용
	- 데이터 타입 사용 위치
		- 변수 타입(멤버 변수 : heap에 저장, 로컬 변수 : stack에 저장) / 메소드 반환 타입
